## 5. Aan de slag: Je eerste code!

### Praktijkvoorbeeld: Interactiviteit toevoegen

Laten we direct aan de slag gaan met een werkend voorbeeld.

**Stap 1:** Maak een nieuwe map op je computer genaamd `JavaScript-Oefeningen`

**Stap 2:** Open deze map in Visual Studio Code

**Stap 3:** Maak een nieuw bestand genaamd `mijn-eerste-javascript.html`

**Stap 4:** Type de volgende code:

```html
<!DOCTYPE html>
<html lang="nl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mijn eerste JavaScript</title>
</head>
<body>
    <h1>Welkom bij JavaScript!</h1>
    <button onclick="toonBericht()">Klik op mij!</button>
    
    <script>
        function toonBericht() {
            alert("Hallo! Dit is JavaScript in actie!");
        }
    </script>
</body>
</html>
```

**Stap 5:** Sla het bestand op en open het in je webbrowser

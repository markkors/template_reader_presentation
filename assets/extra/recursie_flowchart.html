Wat is Recursie?

Een functie die zichzelf aanroept om een probleem op te lossen.

Voorbeeld: 5! = 5 √ó 4 √ó 3 √ó 2 √ó 1 = 120

function faculteit(n) { if (n === 0) { return 1; // Stop hier! } return n * faculteit(n - 1); // Roep jezelf aan }
